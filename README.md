# 챌린지 매니저 플랫폼 MVP

AI 동화 챌린지, 시화 챌린지, 블로그 챌린지, 인스타 챌린지를 운영하기 위한 MVP 웹앱입니다.

## 주요 기능

- 사용자/관리자 로그인
- 챌린지 목록 및 상세 확인
- AI 동화 22일 미션 구조
- 일일 제출, 링크, 캡처 이미지, PDF 제출 관리
- 관리자 제출 검수: 승인, 반려, 추가 확인
- 챌린지 랭킹 계산
- 승인된 제출물 좋아요 반영
- 상금/페이백 계좌 등록 및 관리자 승인
- 보상 지급 상태 관리

## 실행 방법

처음 한 번만 패키지를 설치합니다.

```powershell
npm install
```

개발 서버를 실행합니다.

```powershell
npm run dev
```

브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:5173/
```

## 데모 계정

사용자 계정:

```text
user@demo.com / user123
```

관리자 계정:

```text
admin@demo.com / admin123
```

## 현재 MVP 범위

현재 버전은 실제 결제, 자동 송금, 외부 조회수 연동, AI 동화 자동 생성 기능을 포함하지 않습니다.
챌린지 운영, 제출, 검수, 랭킹, 계좌 등록, 페이백 상태 관리 흐름을 먼저 검증하는 MVP입니다.
