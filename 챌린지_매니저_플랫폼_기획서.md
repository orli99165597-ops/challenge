# 챌린지_매니저_플랫폼_기획서

## 1. 서비스 개요

### 서비스명

챌린지 매니저 플랫폼

### 서비스 한 줄 소개

챌린지 매니저 플랫폼은 사용자가 AI 동화 챌린지, 블로그 챌린지, 유튜브 챌린지에 참여하고, 기간 내 콘텐츠를 완성하면 랭킹 보상과 페이백을 받을 수 있는 참여형 콘텐츠 성장 플랫폼입니다.

### 핵심 챌린지

- AI 동화 챌린지
- 블로그 챌린지
- 유튜브 챌린지

## 2. 서비스 컨셉

사용자는 정해진 기간 동안 챌린지 미션을 수행하고, 결과물을 플랫폼에 제출합니다. 플랫폼은 제출 여부, 콘텐츠 품질, 사용자 반응, 조회수, 댓글 수 등을 기준으로 랭킹을 산정하고 보상을 지급합니다.

챌린지는 단순 참여가 아니라 **매일 수행해야 하는 목표**를 중심으로 설계합니다. 예를 들어 AI 동화 챌린지는 하루에 2쪽, 즉 2페이지씩 작성하는 방식으로 운영할 수 있습니다.

## 3. 챌린지를 하는 이유

챌린지를 운영하는 가장 큰 이유는 사용자가 콘텐츠를 끝까지 완성하도록 돕기 위해서입니다.

AI 동화 챌린지의 핵심 목적은 단순히 글을 몇 페이지 작성하는 것이 아니라, 최종적으로 **AI 동화책 POD 1권을 완성하는 것**입니다. 사용자는 20일 동안 매일 2쪽씩 동화 본문을 만들고, 21일째 전체 페이지를 업스케일하며, 22일째 PDF 제작과 수정, 출판사 등록 준비를 거쳐 실제 출판 가능한 형태의 동화책 한 권을 완성합니다.

블로그 챌린지와 유튜브 챌린지도 같은 철학을 가집니다. 사용자가 막연히 콘텐츠를 시작하는 데서 끝나는 것이 아니라, 정해진 기간 동안 결과물을 쌓고 제출하며, 자신의 콘텐츠 자산을 완성하도록 돕는 것이 목적입니다.

## 4. 서비스 목표

1. 사용자가 콘텐츠 제작을 시작하고 완성하도록 돕는 챌린지 환경 제공
2. AI 동화책 POD 1권 완성을 위한 단계별 제작 흐름 제공
3. AI 동화, 블로그, 유튜브 콘텐츠 제작 습관 형성
4. 챌린지 랭킹과 페이백을 통한 참여 동기 부여
5. 사진, 링크, 영상 URL을 활용한 제출 인증 기능 제공
6. 상금 및 페이백 지급을 위한 계좌 등록 기능 제공
7. 개인 창작자, 부모, 블로거, 유튜버, 교육기관이 함께 활용 가능한 플랫폼 구축

## 6. 핵심 챌린지 유형

## 6.1 AI 동화 챌린지

사용자가 AI를 활용해 동화를 만들고, 정해진 기간 동안 매일 일정 분량을 완성하는 챌린지입니다. 이 챌린지의 최종 목적은 **AI 동화책 POD 1권 완성**입니다. 기본 진행 구조는 20일 동안 동화 본문을 제작하고, 21일째 전체 페이지를 업스케일한 뒤, 22일째 PDF 제작과 부족한 부분 수정, 출판사 등록 준비를 진행하는 방식입니다.

### 일일 수행 기준 예시

- 1일 2쪽 작성
- 1일 2페이지 완성
- 매일 밤 11시 59분까지 당일 분량 제출
- 각 페이지에 제목, 본문, 장면 설명 포함
- 선택 사항으로 페이지별 삽화 또는 사진 포함
- 20일 동안 총 40쪽 분량의 동화 초안 완성

### 진행 기간 및 단계

| 일차 | 단계 | 수행 내용 |
| --- | --- | --- |
| 1일차-20일차 | 동화 제작 | 매일 2쪽씩 작성하고 밤 11시 59분까지 당일 분량 제출 |
| 21일차 | 전체 페이지 업스케일 | 전체 페이지의 삽화, 이미지, 문장 표현, 페이지 구성을 업스케일하고 품질 개선 |
| 22일차 | PDF 제작 및 출판 준비 | 최종 PDF 제작, 부족한 부분 수정, 표지/목차/저자 정보 정리, 출판사 등록 준비 |

### 22일차 출판 준비 항목

- 최종 PDF 파일 생성
- 오탈자 및 부족한 문장 수정
- 페이지 순서 점검
- 표지, 제목, 저자명 확인
- 작품 소개글 작성
- 출판사 등록에 필요한 기본 정보 정리
- 제출용 파일명과 버전 관리

### 주요 기능

- AI 동화 생성
- 캐릭터, 배경, 교훈, 분위기 입력
- 사진 업로드
- 링크 첨부
- AI 삽화 생성
- 동화 편집
- 페이지 단위 작성 관리
- 전체 페이지 업스케일
- 최종 PDF 제작
- 출판사 등록 준비 자료 정리
- PDF 저장
- 작품 피드 공개
- 사용자 투표 및 랭킹 반영

### 제출 방식

- 플랫폼 내부 동화 편집기로 제출
- PDF 파일로 제출
- 페이지별 이미지 또는 캡처 제출
- 참고 사진 및 링크 첨부
- 22일차 최종 PDF 제출

### 평가 기준 예시

- 기간 내 제출 여부
- 매일 밤 11시 59분까지 제출 여부
- 1일 2쪽 작성 기준 충족 여부
- 전체 페이지 수 충족 여부
- 21일차 전체 페이지 업스케일 완료 여부
- 22일차 최종 PDF 제작 및 수정 완료 여부
- 글자 수 충족 여부
- 주제 적합성
- 사진 또는 삽화 포함 여부
- 링크 정상 접근 여부
- 사용자 반응
- 댓글 수
- 콘텐츠 완성도

### 제출 마감 기준

- AI 동화 챌린지의 일일 제출 마감 시간은 매일 밤 11시 59분입니다.
- 밤 11시 59분 이후 제출한 경우 당일 미션 미완료로 처리할 수 있습니다.
- 당일 미션 미완료 여부는 페이백, 일일 미션 달성 보상, 랭킹 점수에 반영됩니다.

### 챌린지 예시

- 환경을 지키는 동화 챌린지
- 잠들기 전 5분 동화 챌린지
- 가족사진으로 동화 만들기
- 내가 사는 동네를 배경으로 한 동화 만들기

## 7.3 제출 인증 기능

사용자가 챌린지를 실제로 완료했는지 확인하기 위한 기능입니다.

### 인증 방식

- 플랫폼 내부 작품 제출
- 외부 블로그 링크 제출
- 유튜브 영상 링크 제출
- 캡처 이미지 제출
- 작성 후기 제출
- 관리자 검수
- 자동 조건 검사

캡처 이미지 제출은 외부 플랫폼의 작성 완료, 업로드 완료, 조회수, 댓글 수, 좋아요 수 등을 증빙하기 위한 인증 방식입니다.

- 블로그 글 작성 완료 화면 캡처
- 블로그 조회수 또는 방문자 수 화면 캡처
- 블로그 댓글 수 화면 캡처
- 유튜브 영상 업로드 완료 화면 캡처
- 유튜브 조회수, 좋아요 수, 댓글 수 화면 캡처
- AI 동화 페이지 작성 완료 화면 캡처
- PDF 생성 완료 화면 캡처

### 캡처 이미지 검수 기준

- 제출한 캡처 이미지가 챌린지 주제와 관련 있는지 확인
- 작성자 또는 채널 정보가 확인 가능한지 검토
- 제출 날짜 또는 업로드 날짜가 확인 가능한지 검토
- 조회수, 댓글 수, 좋아요 수 등 성과 지표가 보이는지 확인
- 조작 의심 이미지인 경우 관리자 검수 대상으로 분류

## 7.5 페이백 시스템

사용자가 챌린지를 완성하고 제출하면 조건에 따라 페이백을 받을 수 있습니다.

### 페이백 지급 조건 예시

- 기간 내 콘텐츠 제출
- 필수 제출 항목 충족
- 일일 수행 기준 충족
- 링크 정상 접근 가능
- 운영 정책 위반 없음
- 중복 또는 표절 콘텐츠가 아님
- 챌린지별 최소 기준 충족

### 페이백 방식

- 참가비 일부 환급
- 플랫폼 포인트 지급
- 다음 챌린지 참가권 지급
- AI 생성 크레딧 지급
- 상위 랭킹 추가 보상 지급
- 현금성 상금 지급

### 페이백 구조 예시

| 구분 | 조건 | 보상 |
| --- | --- | --- |
| 기본 완성 보상 | 기간 내 콘텐츠 제출 완료 | 참가비 일부 페이백 또는 포인트 지급 |
| 일일 미션 달성 보상 | 일일 수행 기준 100% 달성 | 추가 포인트 또는 크레딧 |
| TOP 30% | 랭킹 상위 30% | 추가 포인트 |
| TOP 10 | 최종 랭킹 10위 이내 | 상금 또는 프리미엄 이용권 |
| 특별상 | 창의성, 성장성, 인기상 등 | 별도 보상 |
| 연속 참여 보상 | 3회 이상 챌린지 완성 | 추가 크레딧 또는 참가권 지급 |

## 7.6 상금 수령 계좌 등록 기능

사용자가 챌린지 상금, 현금성 페이백, 환급금을 받기 위해 본인 명의의 계좌 정보를 등록하는 기능입니다.

### 등록 항목

- 예금주명
- 은행명
- 계좌번호
- 생년월일 또는 본인 확인 정보
- 연락처
- 지급 동의 여부
- 개인정보 수집 및 이용 동의

### 주요 기능

- 계좌 정보 등록
- 계좌 정보 수정
- 계좌 인증 상태 표시
- 상금 지급 예정 내역 확인
- 지급 완료 내역 확인
- 지급 실패 사유 확인
- 관리자 검수 상태 확인

### 운영 기준

- 상금 수령 계좌는 본인 명의 계좌 등록을 원칙으로 합니다.
- 미성년자 사용자는 보호자 계좌 등록 또는 별도 동의 절차가 필요할 수 있습니다.
- 계좌 정보는 암호화하여 저장합니다.
- 계좌 정보가 없거나 인증되지 않은 경우 현금성 보상 지급을 보류합니다.

## 10. 수익 모델

수익화 가능성은 있습니다. 다만 광고 수익이나 외부 플랫폼 수익 배분에 바로 기대기보다, 초기에는 **유료 챌린지 참가비, 교육/코칭 패키지, 브랜드 협업 챌린지, POD 출판 준비 서비스**를 중심으로 검증하는 것이 현실적입니다.

## 10.1 챌린지 참가비

사용자는 유료 챌린지에 참가비를 내고 참여합니다. 챌린지를 완성하면 일부 페이백을 받고, 상위 랭킹에 오르면 추가 보상을 받을 수 있습니다.

### 참가비 모델 예시

- AI 동화책 22일 완성 챌린지
- 블로그 7일 글쓰기 챌린지
- 유튜브 쇼츠 7일 제작 챌린지
- 프리미엄 피드백 포함 챌린지

### 운영 방식

- 참가비를 받습니다.
- 완주자는 포인트, 할인권, 일부 페이백을 받습니다.
- 상위 랭킹자는 상금 또는 프리미엄 혜택을 받습니다.
- 미완주자의 참가비 일부는 운영비와 상금 재원으로 활용할 수 있습니다.

## 10.2 프리미엄 구독

### 유료 기능 예시

- 블로그 제목 및 키워드 추천
